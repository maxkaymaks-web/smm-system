# Рабочее место SMM — Инструкция

Привет! Здесь всё что нужно для ежедневной работы с постами.

---

## Ежедневный ритм

### Утром — получить новые задачи

```bash
./tools/sync.sh pull
```

После этого смотри папку `projects/[клиент]/posts/inbox/` — там новые черновики от агентов.

### В течение дня — работа с постами

Используй скиллы в папке `girl-workspace/skills/`. Чтобы применить скилл — просто скажи Claude что нужно сделать.

| Ситуация | Что говорить Claude |
|----------|---------------------|
| Новый пост в inbox | «Проверь пост post-01 проекта [клиент]» |
| Заказчик дал правки | «Запиши фидбек по post-01: [текст правки]» |
| Пост готов к публикации | «Подготовь post-01 к публикации в ВК» |
| Пост плохой | «Верни post-01 на доработку, замечания: [что не так]» |

### Вечером — отправить работу

```bash
./tools/sync.sh push
```

## Структура папок проекта

```
projects/[клиент]/
  posts/
    inbox/      ← 📬 новые посты от агентов (твоя очередь)
    approved/   ← ✅ одобренные тобой
    published/  ← 📢 уже опубликованные
  feedback/     ← 💬 фидбек заказчиков
  context.md    ← 📋 всё о клиенте
```

---

## Файлы каждого поста

Каждый пост состоит из трёх файлов:

| Файл | Что это | Зачем |
|------|---------|-------|
| `post-01.md` | Текст для всех платформ | Копируешь нужный вариант при публикации |
| `post-01.html` | Дизайн-макет | Технический файл, не трогай |
| `post-01.png` | Готовая картинка | Загружаешь при публикации |

---

## Как работать с Claude Code

1. Открой терминал
2. Введи `claude` — запустится Claude Code
3. Скажи что нужно сделать на русском языке

**Примеры команд:**
- «Проверь пост post-01 в inbox проекта client-a»
- «Заказчик хочет убрать хэштеги из post-02, запиши фидбек»
- «Подготовь post-03 к публикации в ВК и Telegram»
- «Перегенерируй PNG для post-01 после правок»

---

## Частые ситуации

**Пост плохой — что делать?**
Скажи: «Верни post-01 на доработку, замечания: [что именно не так]»

**Заказчик хочет правки — что делать?**
Скажи: «Запиши фидбек по post-01: [дословно что сказал заказчик]»

**Не знаешь как звучит бренд клиента?**
Открой `projects/[клиент]/context.md`

**Нужно перегенерировать картинку?**
Скажи: «Перегенерируй PNG для post-01 проекта [клиент]»
