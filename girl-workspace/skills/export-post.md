# Скилл: Экспорт поста для публикации

## Когда использовать

Пост одобрен (в `posts/approved/`) и готов к публикации.

## Что брать из файлов

Открой `posts/approved/post-NN.md`:

| Платформа | Откуда брать текст |
|-----------|-------------------|
| ВКонтакте | Основной текст (первый раздел после `---`) |
| Telegram | Раздел `### Telegram` |
| MAX | Раздел `### MAX` |

Картинка для всех платформ: `posts/approved/post-NN.png`

## Что делать

1. Открой `post-NN.md`, скопируй нужный текст для платформы
2. Возьми `post-NN.png` как картинку
3. Опубликуй вручную в соцсети
4. После публикации перемести файлы:

```bash
mv posts/approved/post-NN.md posts/published/
mv posts/approved/post-NN.html posts/published/
mv posts/approved/post-NN.png posts/published/
```

5. Обнови `content-plan.md` — поставь дату публикации в колонку Статус

## Адаптации для платформ

| Платформа | Длина текста | Хэштеги | Особенности |
|-----------|-------------|---------|-------------|
| ВКонтакте | 300-2200 симв | 3-5 | Первые 2 строки — крючок (видны без раскрытия) |
| Telegram | 300-4000 симв | 0-2 | Эмодзи работают лучше, ссылки не показывают превью |
| MAX | 300-2200 симв | 3-5 | Аналогично ВК |
