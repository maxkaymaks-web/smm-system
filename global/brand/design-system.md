# bit&pix — Дизайн-система агентских документов

> Все внутренние документы (контент-планы, рубрикаторы, сценарии, отчёты)
> создаются в этом стиле.

---

## Цвета

| Переменная | HEX | Назначение |
|---|---|---|
| `--bg` | `#FEFEFF` | Основной фон страницы |
| `--bg-alt` | `#F6FAFC` | Вторичный фон (карточки, шапки таблиц) |
| `--text` | `#0B0B0B` | Весь текст, обводки |
| `--accent` | `#FF1E1E` | Акценты, ключевые элементы, лого |
| `--border` | `rgba(11,11,11,0.10)` | Линии, рамки |

## Шрифт

- **Основной:** Nata Sans (установлен локально)
- **Fallback:** Inter → -apple-system → sans-serif
- Подключение: `font-family: 'Nata Sans', 'Inter', -apple-system, sans-serif`
- Google Fonts fallback: `@import url('https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700;800&display=swap')`

## Радиусы скругления

| CSS var | Значение | Где |
|---|---|---|
| `--radius` | `12px` | Карточки, блоки, обёртки таблиц |
| `--radius-sm` | `6px` | Бейджи форматов, мелкие элементы |
| `--radius-pill` | `20px` | Теги рубрик |

## Логотип

- **SVG inline:** `global/brand/logo.svg` — рекреация для использования в HTML
- **PNG оригинал:** `global/brand/logo.png` — положить сюда оригинал от дизайнера
- Логотип всегда в правом верхнем углу документа

## Теги рубрик (стандартные)

| Рубрика | CSS класс | Стиль |
|---|---|---|
| Технология | `tag-tech` | Красный tint bg, тёмно-красный текст |
| Команда | `tag-team` | Чёрный bg, белый текст |
| Кейс | `tag-case` | Светло-серый bg, чёрный текст |
| Продукт | `tag-product` | Красный bg, белый текст |
| Отрасль | `tag-industry` | Белый bg, серая обводка |
| НИОКР / R&D | `tag-rd` | Белый bg, красная обводка |
| Событие | `tag-event` | Чёрный bg + красная точка |
