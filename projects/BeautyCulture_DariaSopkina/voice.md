## 7 правил голоса

1. **Тёплая забота, не продажа.** Всегда о клиенте, его комфорте, его результате. Без давления.
   - ✅ «чтобы каждая процедура проходила комфортно для вас»
   - ✅ «заботу о теле, которую вы почувствуете уже с первого сеанса»
   - ❌ «купи сейчас», «не упусти», «осталось мало мест»

2. **Обращение «вы» строчная.** Уважительно, но не официально.
   - ✅ «чтобы вы могли получать ещё больше заботы»
   - ❌ «ты», «Вы» с заглавной (исключение: поздравления сотрудникам)

3. **Заголовки — CAPS, без смайлов.**
   - ✅ «КАК МЫ ГОТОВИМ КАБИНЕТ?», «У НАС ОТЛИЧНЫЕ НОВОСТИ!»
   - Заголовок есть ~в 60% постов, не обязателен

4. **Ключевое слово бренда — «забота».** В каждом посте — напрямую или синонимом.
   - «забота о теле», «забота, комфорт и уверенность», «профессиональный уход»

5. **Польза через ощущение, не через цифры.**
   - ✅ «почувствуете уже с первого сеанса», «уверенный и красивый образ жизни»
   - ❌ «-5 см за 10 сеансов», «98% клиентов довольны»
   - Исключение: технические/образовательные посты — там допустимы факты

6. **Команда живая, с именами.** Мастеров называют по имени.
   - «мастер Нана», «при записи к мастеру Татьяне», «От всей команды студии»

7. **Благодарность клиентам — коротко и искренне.**
   - «это доверие, которое мы очень ценим», «И это всё благодаря вам»

## Смайлики

| Смайл | Роль | Позиция |
|-------|------|---------|
| ✨ | Главный. Открывает пост или хороший тезис | Начало абзаца или конец предложения |
| 🤍 | Тёплый акцент после тезиса о заботе/результате | Конец абзаца |
| ❤️ | Благодарность, искренность | Только в эмоциональных постах |
| ☺️ | Мягкая радость | Поздравления, благодарности |
| 💌 | Всегда в паре с «Запись через администратора» | CTA-блок, только там |
| 🔥 | Только для скидок/акций | Редко |
| 🌸 | Акционные заголовки | Редко |

**Правила:**
- Смайл в конце предложения, не в середине
- Не более 1–2 смайлов на абзац
- Заголовок CAPS — без смайлов
- Не использовать: 💥😍🎊🤩 (слишком хайповые)

## Структура поста

```
[ЗАГОЛОВОК CAPS]           ← в ~60% постов, без смайлов

[Первый абзац]             ← суть, приглашение, новость (1–3 строки)
Смайл ✨ в начале или 🤍 в конце

[Второй абзац]             ← польза, детали, список (1–3 строки)
(буллеты через • или ✨ если список)

Запись по ссылке: https://b539491.yclients.com

Запись через администратора 💌
```

**Исключения:**
- Образовательный/технический пост — CTA-ссылка не нужна
- Поздравительный пост — CTA не нужен вообще

## CTA — точная формула (не менять!)

```
Запись по ссылке: https://b539491.yclients.com

Запись через администратора 💌
```

Вариация для постов с адресом:
```
📍 Рубинштейна, 6 | м. Владимирская
💌 Запись через администратора или по ссылке — https://b539491.yclients.com
```

## Характерные фразы (использовать в ротации)

- «забота о себе / о теле»
- «уже с первого сеанса»
- «именно вам» — «подойдёт именно вам»
- «это не просто [X], а [Y]» — приём переопределения ценности
- «благодаря вам»
- «в одном пространстве»
- «заметный результат»
- «Для нас важно, чтобы вы...»

## Запрещено — никогда

| Нельзя | Почему |
|--------|--------|
| Хэштеги | Полный запрет — решение Дарьи |
| Агрессивные триггеры | «осталось 2 места», «только сегодня», «не упусти» |
| «ты» | Только «вы» |
| Числовые обещания | «-10 см», «за 5 сеансов», «100% эффект» |
| Сленг | «топ», «огонь», «красотка», «пушка» |
| Восклицательные знаки в теле поста | В заголовках — редко и уместно |
| Скидки и акции | Только с явного одобрения Дарьи |
| Сравнения: RSL vs LPG, лазер vs шугаринг | Запрещено |
| Дарья лично в контенте | Запрещено |
