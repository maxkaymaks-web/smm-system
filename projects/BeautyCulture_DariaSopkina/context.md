## Визуальный стиль

- **Размер холста:** 1080×1350px (портрет — стандарт для каруселей)

- **Основной светлый фон:** #F5F0EA (кремовый)
- **Тёмный фон:** #1a1714
- **Текст на тёмном:** белый #FFFFFF
- **Стиль:** минималистичный, плёночная эстетика, grain-текстура
- **Логотип «Культура»:** top-right, мелким шрифтом
- **Нумерация слайдов:** bottom-right, формат 01/02/03
- **CTA-слайд:** всегда тёмный финальный слайд
- **Nano Banana watermark:** удалять обрезкой 180px снизу (при вставке их фото)

## Дизайн-предпочтения Дарьи

Зафиксировано по итогам правок карусели «Комплексы S/M/L» (апрель 2026).

### ✅ Нравится

- Крупная типографика: буквы S/M/L/XL как главный визуальный элемент
- Минималистичный кремовый фон (#FAF8F5)
- Нумерация зон (01/02/03) — читаемо и структурированно
- Editorial full-bleed фото модели как фон (вместо before/after) — **одобрено апрель 2026**
- Ghost letter (большая полупрозрачная буква комплекса) — одобрено
- Тёмный CTA-финальный слайд
- Grain texture (плёночная эстетика)
- Декоративные ботанические SVG-элементы (тонкие линии)
- Крупная цена по центру снизу без конкурирующих элементов

### ❌ Не нравится / полный запрет

| Элемент | Почему запрещено |
|---------|-----------------|
| **SVG-схема тела («человечек»)** | Выглядит дёшево, не в стиле студии — **никогда не использовать** |
| Пустые зоны между блоками | Пространство должно быть заполнено или убрано |
| **Before/after в карусели услуг** | Заменить на editorial фото — одобрено апрель 2026 |
| **Filled pills поверх фото** | Смотрятся тяжело, Дарья попросила убрать — не использовать |

### Правило для цены

Цена — главный визуальный акцент нижней части слайда:
- По центру
- Максимально крупно (от 130px)
- Без конкурирующих элементов рядом (никаких иллюстраций / схем)
