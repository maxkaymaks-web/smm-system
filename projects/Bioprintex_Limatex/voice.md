## 6 правил голоса

1. **Только факты и цифры — никакой воды.** Каждый тезис должен быть подкреплён данными.
   - ✅ «0,8 руб/м³ кислорода против стандартных 3–5 руб/м³»
   - ✅ «плотность посадки рыбы ×3, энергопотребление −25%»
   - ❌ «инновационные решения», «передовые технологии», «мы лучшие в отрасли»

2. **Экспертный тон — говорим как профессионал с профессионалом.** Аудитория — технари и управленцы промышленных предприятий. Не объяснять очевидное, не упрощать до потери смысла.
   - ✅ «газоперенос по Henry-закону», «ХПК ниже нормативов на 40%»
   - ❌ «вода становится чище», «рыбам лучше дышится»

3. **Конкретная экономика.** ROI, сроки окупаемости, снижение OPEX — обязательны в продуктовых постах.
   - ✅ «окупаемость — 18 месяцев», «CAPEX −30–50% через заказной R&D»
   - ❌ «значительная экономия», «быстрая окупаемость»

4. **Без эмоционального давления.** Не «купите», не «срочно», не «уникальное предложение». Факты сами продают.

5. **Обращение на «вы».** Нейтральное, деловое. Без панибратства.

6. **CTA — конкретное действие.** Не «подпишитесь», а «обсудить задачу» / «получить расчёт» / «запросить КП».

## Смайлики

Используются **редко и точечно** — только если пост не технический.
- Технические и продуктовые посты: смайлы не нужны
- Команда / новости / события: допустимы 1–2 нейтральных

Разрешены: 🔬 💧 🌿 📊 — по теме
Запрещены: ❤️ 🤍 ✨ 🔥 — ассоциируются с B2C, не подходят

## Структура поста

### Экспертный / Технологический
```
[Цепляющий факт или вопрос]   ← первая строка = главный тезис

[Объяснение / контекст]       ← 2–4 строки, техника или экономика

[Вывод с цифрой]              ← итоговый результат / применимость

[CTA]                         ← конкретное действие
```

### Продуктовый
```
[Проблема клиента]            ← боль отрасли, цифры потерь

[Как решает продукт]          ← механика + параметры

[Экономика]                   ← ROI, сроки, снижение затрат

[CTA]
```

### Отраслевой
```
[Контекст / тренд]            ← данные по отрасли

[Связь с нашей экспертизой]   ← почему мы в теме

[CTA]
```

## Запрещено — никогда

| Нельзя | Альтернатива |
|--------|-------------|
| «инновационный», «уникальный», «революционный» | Конкретный параметр продукта |
| «лучший в отрасли», «№1» | «соответствует ГОСТ», «патент №...» |
| Эмоциональные обещания без цифр | Цифра + источник |
| Хайповые смайлы (❤️✨🔥) | Без смайлов или тематические |
| Упрощение технических терминов до неточности | Точный термин + краткое пояснение |
| Скрытые допущения в цифрах | Указывать условия: «при плотности 50 кг/м³» |
