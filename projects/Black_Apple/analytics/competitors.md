## Рекомендации для Black Apple

### Приоритет 1 — Исправить MSK: геолокация в каждом посте

**Проблема:** Москва не прописана в тексте постов → алгоритм не видит геолокацию → нет локального таргетинга.

**Решение:** В каждый продающий пост добавлять:
```
Москва. Black Apple | [модель]
📍 [адрес]
```
Это единственная разница между 95v (MSK) и 760v (Kursk) при одинаковых форматах.

### Приоритет 2 — Прайс б/у с техническими деталями

Формат Курска работает: `модель — объём — цвет — е-сим/нано — % АКБ — цена`. Не «iPhone 13 б/у», а:
```
🍏 iPhone 13 256 черный 100% АКБ 28 500₽
🍏 iPhone 14 Pro Max 128 фиолет е-сим 84% АКБ 44 000₽
```
Покупатель видит всё нужное без переписки → выше конверсия.

### Приоритет 3 — «Проблема + решение» по модели @re_store

Самый виральный формат ниши (54k). Адаптация для продающего аккаунта:
- «Купил б/у iPhone — вот как проверить, не вскрывался ли он»
- «Рассрочка на iPhone: что скрывают менеджеры магазинов»
- «iOS X вышла — стоит ли обновляться? Что мы проверили»

Такой контент массово репостится → рост аудитории → рост продаж.

### Приоритет 4 — Карусель как основной формат (оба города)

Оба аккаунта: карусель доминирует выше медианы (17 из 25 постов выше медианы — каруселей). Видео — высокая частота, но не лучший охват. Вывод: **карусель = главный формат для продающего контента**.

Оптимальная карусель:
- Слайд 1: заголовок + модель + цена крупно
- Слайды 2-5: фото устройства с разных сторон
- Слайд 6: условия (рассрочка, гарантия, адрес)

### Приоритет 5 — Рубрики для регулярного охвата

| Рубрика | Частота | Механика |
|---------|---------|---------|
| **Прайс недели** | 3×/нед | Актуальный каталог б/у с ценами, Москва/Курск в тексте |
| **Совет по iPhone** | 1×/нед | Проблема → решение (по модели @re_store) |
| **Сравнение** | 1×/нед | iPhone vs конкурент, новая vs старая модель |
| **Акция/скидка** | 2×/нед | Конкретная модель + условия (нал/рассрочка) |

### Визуальный стиль — что делать и не делать

**Делать:**
- Фирменный цвет + логотип на каждом слайде каруселей
- Фото устройств на нейтральном (чёрном/белом) фоне
- Крупный ценник на первом слайде читается без открытия поста

**Не делать:**
- Посты без текста (6k просмотров — исключение, не правило)
- Стоковые фото из интернета без брендинга
- Искажённый текст / замена букв — угроза блокировки
