## Рекомендации для @lis.gym

### Приоритет 1 — Формула рилса (немедленно)

**Hook в первые 2 секунды.** Сейчас: «gym therapy>>>>>» — нет крючка. Должно быть: вопрос / утверждение / ситуация, с которой зрители себя узнают.

Примеры крючков для зала:
- «Один лайфхак, который изменит твою тренировку ягодиц» → образование
- «Это зал, куда ты захочешь вернуться» → атмосфера
- «Почему ты не видишь прогресс, хотя ходишь в зал 3 раза в неделю» → боль

### Приоритет 2 — Длина (немедленно)

Целевая длина: **6-12 секунд** для виральных рилсов, 15-20s для образовательных.
Текущие 18.4s — верхняя граница. Сжать.

### Приоритет 3 — Хэштеги (переосмыслить)

- Убрать generic: #gymgirl #gym #gymmotivation — не конвертируют
- Оставить: #санктпетербург + гео-теги района + название зала
- Формат @km.ksss: #санктпетербург #тренажерка #зал — 111k на relatable посте
