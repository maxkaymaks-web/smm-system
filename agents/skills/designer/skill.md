# Агент: Дизайнер

Вызывается оператором постов для создания визуала поста (HTML → PNG).

---

## Алгоритм вызова

1. Убедись что `post.md` с текстом уже готов
2. Прочитай дизайн-параметры проекта (ниже)
3. Сформируй ТЗ по шаблону
4. Запусти агента через Agent tool
5. Результат: `post.html` + `post.png` в той же папке что `post.md`
6. git commit + push

---

## Дизайн-параметры проектов

### BeautyCulture_DariaSopkina
- Размер: 1080×1350px (портрет)
- Шрифт заголовков: Cormorant Garamond
- Шрифт текста: Raleway
- Светлый фон: #F5F0EA, тёмный фон: #1a1714
- Стиль: минималистичный, плёночная эстетика
- Логотип «Культура»: top-right, мелко

### Bioprintex_Limatex
- Размер: 1080×1080px (квадрат) или 1080×1350px (портрет)
- Стиль: строгий, технический, B2B
- Минимум декора, акцент на данные и текст

---

## Шаблон ТЗ для дизайнера

```
Ты — дизайнер агентства bit&pix. Создай визуал для поста.

Проект: {ProjectID}
Дизайн-параметры: [вставить параметры проекта выше]

Текст поста:
[вставить содержимое post.md]

Тип: [одиночный пост / карусель]
Слайдов: [N если карусель]

Сохранить:
- Одиночный: post.html + post.png
- Карусель: slide_01.html + slide_01.png, slide_02.html + ...

Рендер командой:
node tools/render-html.js [путь к .html] [путь к .png]
```

---

## Важно

- Рендер требует Node.js — убедись что установлен (`node --version`)
- Если карусель — каждый слайд отдельный HTML + PNG
- Текст на изображении: максимум 20% площади
- Отступы от краёв: минимум 60px
