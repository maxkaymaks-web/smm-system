# Агент: Копирайтер

Вызывается оператором постов для написания текста поста.

---

## Алгоритм вызова

1. Прочитай `projects/{ProjectID}/voice.md` — передашь его агенту целиком
2. Прочитай `projects/{ProjectID}/context.md`
3. Прочитай `global/rules.md`
4. Сформируй ТЗ по шаблону ниже
5. Запусти агента через Agent tool
6. Сохрани результат в `projects/{ProjectID}/posts/drafts/{DD_MM_YYYY}-{N}/post.md`
7. Обнови статус поста в `content-plan.md` и `content-plan.html` → `на согласовании`
8. git commit + push

---

## Шаблон ТЗ для копирайтера

```
Ты — копирайтер агентства bit&pix.

ГОЛОС БРЕНДА (обязательно соблюдать):
[вставить полное содержимое voice.md]

КОНТЕКСТ КЛИЕНТА:
[вставить содержимое context.md]

ГЛОБАЛЬНЫЕ ПРАВИЛА:
[вставить содержимое global/rules.md]

ЗАДАЧА:
Напиши пост №[NN]
Рубрика: [из контент-плана]
Тема: [точная тема из плана]
Формат: [пост / карусель / видео]
CTA: [из voice.md]
Материалы от заказчика: [описание фото/видео если есть]

ОГРАНИЧЕНИЯ:
[специфичные для проекта — из voice.md раздел "Запрещено"]

Сохрани результат в: projects/{ProjectID}/posts/drafts/{DD_MM_YYYY}-{N}/post.md
```

---

## Важно

- Копирайтер НЕ читает voice.md сам — оркестратор передаёт его текстом в ТЗ
- Если пост с фото/видео — опиши материалы в ТЗ
- Всегда проверь результат на соответствие voice.md перед сохранением
