### Beauty Aesthetics Clinic — подборка 12 постов (Instagram)
**Ниша:** beauty / aesthetics / инъекционная косметология
**Применимо к:** BeautyCulture и аналогичные студии красоты

**Ключевые приёмы:**

1. **Крупный кроп как фон** — фото не иллюстрирует, а заполняет. Губы на весь кадр, текстура кожи крупно, закрытый глаз. Никакого "человек на фоне студии".

2. **Типографская пара Script + Bold** — всегда два шрифта в паре. Bold caps = якорь (смысл), Script italic = акцент (настроение). Script меньше Bold примерно в 2 раза по размеру.

3. **Служебные элементы в углах** — бренд top-left, номер/дата top-right, CTA bottom-left или bottom-right. Никогда не меняются между постами — создают узнаваемость серии.

4. **Образовательный контент → split-панель** — как только появляется структурированная информация (таблица, список, шаги), холст делится 50/50: фото слева, белая панель с контентом справа.

5. **Продуктовый пост без людей → editorial** — светлый фон, огромный заголовок сверху, продукт по центру, callouts позиционированием без рамок.

6. **Дизайн-детали в заголовке** — апостроф, дефис, перенос строки используются как визуальные элементы (RET'NOL, "We Are / Open" на двух строках).

7. **Минимум текста поверх фото** — body text мелкий (24-26px), сжатый, занимает не больше 15-20% площади. Фото "дышит".

**Типографика серии:**
- Bold: Inter / Montserrat / Neue Haas — 700-900, uppercase, 60-140px в зависимости от длины
- Script: Cormorant Italic / Playfair Display Italic / Dancing Script — 40-70px
- Служебный текст: 16-20px, caps, letter-spacing 0.1-0.15em

### Shodwe Beauty & Spa — "Our Services" (story)
**Ниша:** beauty / spa / wellness
**Применимо к:** любая студия с меню услуг

**Приёмы:**
- Floating card: округлённая кремовая панель (border-radius ~20px) поверх ambient spa-фото — не закрывает фото полностью, по краям фон виден
- Список услуг: pill-контейнеры border-only, нумерация в circle слева, название caps по центру — без заливки, воздушно
- CTA кнопка: вынесена из карточки вниз, на фоне фото — создаёт ощущение "третьего слоя"
- Декор внутри карточки: маленькая иконка (солнце/логотип) над заголовком — даёт идентичность без лишнего текста
- Sparkle-звёздочки по бокам заголовка — лёгкий акцент, не кричащий

### @earthlyderm — "Repair Skin Barrier" (инфографика)
**Ниша:** skincare / beauty education
**Применимо к:** образовательные посты с 4-6 пунктами

**Приёмы:**
- Cross-grid: крест из тонких белых линий делит портретный холст на 6 зон (2×3) — чистая геометрия прямо поверх фото кожи
- Sparkle на пересечениях линий — минимальный декор, не перегружает
- Outline SVG-иконки белые — работают на любом фоне, не "тяжёлые"
- Заголовок вынесен вверх над крестом, отдельно от сетки
- Фото кожи крупным кропом: тёмная сторона шеи/плеча создаёт нужный контраст для белого текста

### Moon Laser Clinic — карусель "Гормонозависимые зоны" (4 слайда)
**Ниша:** beauty / laser clinic / медицинская косметология
**Применимо к:** образовательные карусели в клинической нише

**Приёмы:**
- **Pill-tag pair** в углах каждого слайда (`moon` | `laser`) — идентификатор серии вместо логотипа
- **Ритм white→photo→photo→white**: открывающий белый (тема) → два фото-слайда (детали/зоны/объяснение) → закрывающий белый (вывод/CTA)
- **Ghost watermark**: "MOON" крупно, opacity ~0.09, нижняя треть фото-слайдов
- **Symbol accent**: знак `≠` между двумя headline на белом слайде — типографика вместо иконки
- **Bracketed disclaimer**: `[ Поэтому если... ]` — квадратные скобки дают клинический тон
- **Outline SVG-иллюстрация** (волосяной фолликул, тонкие линии) на белых слайдах — единственный визуальный элемент, не перегружает
- **Bullet list на фото** без панели — работает за счёт тёмной зоны фото в верхней части

### Feyablanche_ — серия stories (личный бренд / lifestyle)
**Ниша:** personal brand / lifestyle / coaching
**Применимо к:** авторские серии, образовательные карусели

**Приёмы:**
- **Два режима фона** чередуются: тёмная органическая текстура (шоколад/глина) для брендовых слайдов ↔ светлый чистый фон для контентных
- **Mood-слайд без текста** — чёрно-белое fashion-фото как пауза в серии
- **Фиксированный footer** из 3 тег-пилюль — якорь серии, одинаковый на каждом слайде, меняется только цвет (белый на тёмном, тёмный на светлом)
- **Checklist-раскладка**: список с круглыми иконками слева, мелкий body text под каждым пунктом, фото тела снизу как визуальный якорь
- **Strikethrough list** — зачёркнутые пункты как приём ("I don't need"), визуально интереснее обычного списка
- **Story нумерация** top-right ("STORY 4/4") — читатель понимает место в серии
