### Full-bleed фото + текст поверх
**Когда:** beauty / aesthetics — промо, анонс, продукт
**Суть:** Фото занимает весь холст (1080×1080 или 1080×1350). Текст накладывается поверх без подложки — работает за счёт контраста (светлый текст на тёмном участке фото или наоборот).
**Объекты:**
- Фото: `position:absolute; inset:0; background-size:cover; background-position:center`
- Заголовок: низ или середина слева, `position:absolute; left:60px; bottom:120px`
- Бренд + категория: мелко в углах сверху (`top:40px; left:60px` и `top:40px; right:60px`)
- CTA: мелко снизу, противоположный угол от заголовка
**Правило фото:** только крупный кроп — губы, кожа, часть лица. Никогда полный портрет. Создаёт intimacy без перегруза.

### Script + Bold Sans — типографская пара
**Когда:** beauty / lifestyle — любой тип поста
**Суть:** Главное слово — bold uppercase sans (визуальный якорь). Второстепенное — italic script (смягчение, контраст). Script никогда не несёт ключевой смысл — только акцент.
**Реализация в HTML:** два отдельных элемента, script через Google Fonts (Playfair Display Italic / Dancing Script / Cormorant Italic), sans — Inter/Montserrat Bold. Script располагается над или под bold-словом, меньший размер (≈40-50% от bold).
**Пример:** `<div class="script">Special Promo</div><div class="bold">FILLER BUNDLE</div>`
---

### Editorial layout — светлый фон, продукт как герой
**Когда:** beauty — продуктовый spotlight, без фото людей
**Суть:** Кремовый фон, огромный заголовок занимает 30-40% высоты холста, продукт (PNG без фона) по центру или смещён. Callout-блоки — пользы/характеристики — расположены вокруг продукта без рамок, только позиционированием.
**Объекты:**
- Фон: однотонный, `background: #f5f0ea` или аналог из бренда
- Заголовок: `font-size: 120-140px`, `font-weight: 900`, занимает верх
- Продукт: `position:absolute; центр холста`, высота 400-500px
- Callouts: `position:absolute` вокруг продукта, `font-size: 20-22px`, без декора
**Спецприём:** апостроф или дефис в заголовке как дизайн-элемент (RET'NOL, не RETINOL).
---

### Split — фото 50% + инфо-панель 50%
**Когда:** beauty — образовательный контент, do's & don'ts, сравнения
**Суть:** Холст делится вертикально пополам. Левая половина — фото (full-height). Правая — белая или фирменная панель с таблицей / списком / колонками.
**Реализация в HTML:**
```css
.photo { position:absolute; left:0; top:0; width:50%; height:100%; background-size:cover; }
.panel { position:absolute; right:0; top:0; width:50%; height:100%; padding:60px 48px;
         display:flex; flex-direction:column; justify-content:center; }
```
**Когда НЕ использовать:** если контент не структурированный (нет списка/таблицы) — лучше full-bleed.
---

### Floating card — панель поверх фото
**Когда:** beauty / spa / wellness — список услуг, прайс, меню
**Суть:** Фото занимает весь холст как фон (часто размытый или затемнённый). Поверх — rounded карточка с контентом, занимает 60-70% ширины, центрирована. Создаёт глубину без split-раскладки.
**Объекты:**
- Фон: `position:absolute; inset:0; background-size:cover; filter:brightness(0.85)` опционально
- Карточка: `position:relative; z-index:2; background:#faf8f4; border-radius:24px; padding:48px 40px; margin:auto`
- CTA кнопка: вынесена **вне** карточки, ниже неё — отдельный элемент на фоне фото
- Иконка/логотип: внутри карточки сверху, по центру, небольшая (40-50px)
**Нумерованный список внутри карточки:** pill-контейнеры с `border:1.5px solid #1a1a1a; border-radius:40px; padding:14px 24px` — цифра в circle слева, текст caps по центру. Без заливки.
---

### Cross-grid — крест делит фото на зоны
**Когда:** beauty / skincare — образовательный контент, 4-6 пунктов, инфографика
**Суть:** Два тонких белых перпендикулярных линии (1-1.5px) делят холст на 4 или 6 зон прямо поверх фото. В каждой зоне — outline SVG-иконка + 2-строчная подпись. Пересечения линий украшены sparkle-звёздочкой.
**Реализация в HTML (SVG-линии):**
```html
<!-- Вертикальная линия -->
<div style="position:absolute;left:50%;top:180px;bottom:60px;width:1px;background:rgba(255,255,255,0.6)"></div>
<!-- Горизонтальные линии (для 3 рядов — 2 линии) -->
<div style="position:absolute;top:33%;left:60px;right:60px;height:1px;background:rgba(255,255,255,0.6)"></div>
<div style="position:absolute;top:66%;left:60px;right:60px;height:1px;background:rgba(255,255,255,0.6)"></div>
```
**Иконки:** white stroke SVG, 40-50px, `position:absolute` в центре каждой зоны.
**Подписи:** `color:white; font-size:18-20px; text-align:center; letter-spacing:0.1em; text-transform:uppercase`.
**Когда НЕ использовать:** если иконок нет или меньше 4 пунктов — лучше обычный список.
---

### Texture background — тёмная текстура вместо фото
**Когда:** beauty / lifestyle / personal brand — брендовые слайды серии, философия, анонсы
**Суть:** Фон — тёмная органическая текстура (мокрая глина, кожа, камень, бумага) вместо фото людей или однотонного цвета. Создаёт тактильность и премиальность. Белый текст поверх.
**Реализация в HTML:** CSS `background-image` + SVG feTurbulence (grain) поверх. Или: фото текстуры через `background-image:url(...)` + `filter:brightness(0.6) saturate(0.8)`.
**Когда текстура — grain-overlay уже в skill.md.** Для тёмного текстурного фона: `background:#2a1f1a` + grain overlay opacity 0.12.

### Series footer — тег-пилюли как идентификатор серии
**Когда:** любая ниша — серия постов / stories с единым стилем
**Суть:** Фиксированная строка из 2-4 тег-пилюль в самом низу каждого поста серии. Разделитель между тегами — `|` или `·`. Цвет/форма не меняются от слайда к слайду — это якорь узнаваемости.
**Реализация:**
```css
.series-footer {
  position: absolute; bottom: 40px; left: 0; right: 0;
  display: flex; justify-content: center; gap: 12px;
}
.tag-pill {
  border: 1px solid currentColor; border-radius: 20px;
  padding: 6px 20px; font-size: 16px; letter-spacing: 0.12em;
  text-transform: uppercase;
}
```

### Чередование слайдов в серии (ритм)
**Когда:** stories-серия, карусель из 4+ слайдов
**Суть:** Визуальный ритм строится через чередование типов слайдов. Не все одинаковые — это усталость. Классическая схема: тёмный (бренд) → светлый (контент) → mood-фото (пауза).
**Схема:**
1. **Брендовый** — тёмный фон/текстура, headline, минимум текста
2. **Контентный** — светлый фон, список / инфо / цитата
3. **Mood** — чистое фото без текста или с минимальной подписью (пауза, "дыхание")
4. Повтор или CTA-финал
**Правило:** mood-слайд без текста — обязательно хотя бы раз в серии из 4+.
---

### Pill-tag pair — идентификатор серии в углах
**Когда:** любая ниша — карусель / серия, где нужно связать слайды без повторения логотипа
**Суть:** Два rounded pill-тега (border-only, без заливки) в верхних углах каждого слайда. Содержат короткие слова — название бренда, продукта или категории. Заменяют классический "бренд top-left" и делают серию узнаваемой.
**Реализация:**
```css
.pill { border: 1.5px solid currentColor; border-radius: 40px;
        padding: 8px 20px; font-size: 16px; letter-spacing: 0.05em; }
.pill-left  { position:absolute; top:40px; left:40px; }
.pill-right { position:absolute; top:40px; right:40px; }
```
**Цвет:** белый на фото-слайдах, тёмный на белых слайдах — адаптируется к фону.
---

### Ghost watermark — призрачный бренд на фото
**Когда:** beauty / clinic — фото-слайды в серии
**Суть:** Название бренда крупным шрифтом (150-200px+), `opacity: 0.08-0.12`, в нижней трети фото. Не читается явно — работает как текстура, усиливает присутствие бренда без перегруза.
**Реализация:**
```css
.ghost-brand {
  position: absolute; bottom: 80px; left: 0; right: 0;
  text-align: center; font-size: 180px; font-weight: 900;
  color: white; opacity: 0.09; letter-spacing: 0.05em;
  pointer-events: none; user-select: none;
}
```
**Отличие от botanical (BeautyCulture):** ghost — текст/слово, botanical — SVG-линии. Разный характер.
---

### Symbol accent — математический знак как визуальный якорь
**Когда:** beauty / clinic / education — белые типографские слайды с противопоставлением
**Суть:** Математический или типографский символ (`≠`, `=`, `→`, `×`, `+`) большого размера (80-120px) между двумя headline-ами. Заменяет иконку — никакого SVG не нужно. Работает в клинической / editorial эстетике.
**Пример:** "СЛОЖНЕЕ И ДОЛЬШЕ УБРАТЬ" + `≠` (100px, bold) + "НЕВОЗМОЖНО".
**Реализация:** обычный `<div>` с `font-size:100px; text-align:center; margin:20px 0`.
---

### Bracketed text — скобки как клинический тон
**Когда:** beauty / clinic / medical — важная оговорка, пояснение, disclaimer
**Суть:** Тело текста обёрнуто в квадратные скобки `[ текст ]`. Создаёт ощущение технического/медицинского документа без дополнительного дизайна. Работает на фото и на белом фоне.
**Пример:** `[ Поэтому если у вас есть проблемы на этом фоне, волосы могут расти активнее ]`
**Реализация:** просто добавить `[` и `]` в текст — никаких CSS-рамок.
---

### Карусель white↔photo↔photo↔white (клинический ритм)
**Когда:** beauty / clinic — образовательная карусель 4+ слайдов
**Суть:** Строгое чередование типов фона: белый (типографика) → фото (эмоция) → фото (детали) → белый (вывод). Белые слайды — "вдох", фото — "выдох". Финальный слайд белый = итог + CTA.
**Правило белого слайда:** только headline + один визуальный элемент (иконка SVG или символ). Никаких фото, градиентов, фонов.
**Правило фото-слайда:** текст ТОЛЬКО на тёмной зоне фото — без подложек. Если светлое фото — тёмный текст.
---

### Универсальная анатомия поста (beauty)
**Когда:** любой тип поста в beauty-нише
**Суть:** Фиксированная раскладка служебных элементов, не зависящая от типа контента.
**Схема:**
```
[BRAND NAME — top-left, 16-18px, caps, letter-spacing 0.15em]    [00/00 — top-right]

        [основной контент — центр или низ]

[CTA или рубрика — bottom-left, 18-20px]        [категория — bottom-right]
```
**Реализация:** бренд и категория — `position:absolute`, все четыре угла с отступом 40-50px от краёв. Основной контент — flexbox по центру или абсолютно прибит к низу с отступом 100-120px.
