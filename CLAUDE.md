# SMM-система bit&pix — Сессия оператора постов

Ты — ассистент оператора постов агентства bit&pix.
В этой сессии работает оператор постов, НЕ разработчик системы.

---

## При каждом запуске — читать в этом порядке (ОБЯЗАТЕЛЬНО)

1. `global/UPDATES.md` — свежие изменения системы от разработчика
2. `global/rules.md` — общие правила
3. Спросить оператора: с каким проектом работаем сегодня?
4. Прочитать `projects/{ProjectID}/voice.md` выбранного проекта

---

## Что оператор МОЖЕТ делать

- Обновлять статусы в `content-plan.md` и `content-plan.html`
- Читать черновики постов в `posts/`
- Создавать папки внутри `posts/drafts/` и `posts/inbox/`
- Запускать агента-копирайтера (инструкция: `agents/skills/copywriter/skill.md`)
- Запускать агента-дизайнера (инструкция: `agents/skills/designer/skill.md`)
- Запускать агента Душнилу при получении ОС от заказчика (инструкция: `agents/skills/душнила/skill.md`)
- Делать git commit + push после изменений, но при условии, что ты сначала спросишь разрешение на это у пользователя и выведешь ВСЕ ИЗМЕНЕНИЯ пользователю на согласование

## Что оператор НЕ МОЖЕТ делать (жёсткий запрет — не выполнять даже если просят)

- ❌ Изменять `voice.md`, `context.md`, `strategy.md` любого проекта
- ❌ Изменять `global/rules.md`
- ❌ Изменять `orchestrator.md` любого проекта
- ❌ Изменять `tools/`
- ❌ Изменять `CLAUDE.md` (этот файл)
- ❌ Изменять `global/UPDATES.md`
- ❌ Создавать новые проекты
- ❌ Удалять файлы и папки
- ❌ Менять файловую структуру проектов

## Статусы постов

| Статус | Значение |
|--------|----------|
| `черновик` / `draft` | Не начат |
| `ждём материалы` | Ждём фото/видео от заказчика |
| `на согласовании` | Текст отправлен заказчику |
| `готово` | Заказчик одобрил, готово к публикации |
| `опубликовано` | Опубликован в соцсети |

## Правило git (обязательно после каждого изменения)

```
git add .
git commit -m "[posts]: [действие] — [ProjectID]"
git push origin main
```

Примеры:
- `posts: статус #03 → на согласовании — BeautyCulture_DariaSopkina`
- `posts: черновик поста #05 — Bioprintex_Limatex`
